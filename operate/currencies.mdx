## Social casino

| Code | Name | Symbol | Decimals |
|:--|:--|:--|:--|
| GC | Game Coins | GC | 2 |
| SC | Service Coins | SC | 2 |

GC and SC behave identically to fiat currencies but render with right-aligned text symbols instead of currency glyphs. You can supply custom SVG icons via the `gcCurrencyIconSvgUrl` and `scCurrencyIconSvgUrl` attributes. See [branding](/operate/branding).

## Formatting

- Currencies with 0 decimal places (JPY, CLP) display as whole numbers.
- All other currencies display with 2 decimal places.
- Set `showCurrencyAsText="true"` to render the currency code (e.g. `USD 1,000`) instead of the symbol. Useful for ambiguous-glyph currencies like the dollar.

## Configuration

Pass the code and decimals to the embed:

```html
<origami-game-provider-wrapper
  currency="BRL"
  currencyDecimals="2"
  balance="5000"
  ...
></origami-game-provider-wrapper>
```

## Languages

Games ship localized in 17 locales: Arabic, Chinese, English, English (social casino), Finnish, French, German, Hindi, Indonesian, Japanese, Korean, Polish, Portuguese, Russian, Spanish, Turkish, and Vietnamese. Set the locale via the `language` attribute on the embed.
