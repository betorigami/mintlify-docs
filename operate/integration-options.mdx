If you already integrate with Hub88 or SoftSwiss, adding Origami usually means enabling us as a provider. No new integration work on your side.

## Direct API

Direct integration gives you the most control: every attribute, every event, every branding knob, with no aggregator in the middle.

[Start the direct integration →](mailto:sales@betorigami.com)

## Choosing a path

| | Aggregator | Direct |
|:--|:--|:--|
| **Time to live** | Fastest, usually a provider flip | About a week, depending on your wallet stack |
| **Customization** | Bounded by aggregator feature set | Every embed attribute, every webhook event |
| **Latency** | One additional network hop | Direct connection to Origami |
| **Fees** | Aggregator takes a cut | No intermediary |
| **Branding** | Varies by aggregator | Full white-label |
| **Limits and exposure** | Shared with other providers | You set them |

<Tip>
Use the aggregator you already have unless you need brand-level customization, custom SVG artwork, or per-market configuration that only direct integration exposes.
</Tip>
